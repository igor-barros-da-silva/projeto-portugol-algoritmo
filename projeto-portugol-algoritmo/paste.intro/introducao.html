Algoritmo e Portugol Algoritmo É simplesmente uma "receita" para executarmos uma tarefa ou resolver algum problema. Portugol Também conhecido com Português estruturado, é um pseudocódigo escrito em português. Exemplo Algoritmo "Celsius" Var F,C: REAL Inicio ESCREVAL("Colocar temperatura em Fahrenheit") LEIA(F) C <- (F-32)*(5/9) ESCREVAL("Conversor de temperatura") ESCREVAL("A temperatura é: ",C,"Graus Celsius") Fimalgoritmo IDE: Visualg 3.0
